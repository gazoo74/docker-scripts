== NAME

docker-archive - show archive content

== SYNOPSIS

*docker-archive* [OPTIONS] TAR...

== DESCRIPTION

Display à la git-log the content of docker archive to stdout.

== OPTIONS

**-j or --json**
	Display json content.

**-h or --help**
	Print usage.
